项目打开方式：双击项目文件夹下的Semantic即可

项目功能模块：
1）打开文件      可以将测试文件内容导入 （正确测试，错误测试）
2）词法规则      显示RE和DFA转换表
3）词法分析      显示词法分析结果       （词法单元，DFA转换图，词法错误）
4）语法规则      显示文法分析的结果     （FIRST集，FOLLOW集，SELECT集，预测分析表）
5）语法分析      显示语法分析结果       （文法内容，语法分析树，语法错误）
6）语义规则      显示文法及SDT
7）语义分析      显示语义分析结果       （三地址指令 可以查看生成三地址指令的程序行号，四元式，符号表，语义错误）
8）清空界面      将主界面的程序清空


注意：需按照文法进行程序识别，否则识别结果可能有误。
